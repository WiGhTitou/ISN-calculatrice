Quelques indications

Ce tableau périodique permet d’obtenir des informations sur les éléments en passant simplement sa souris dessus. La calculatrice à droite permet quant à elle, de réaliser bien entendu des calculs basiques, mais aussi des formules de chimie. Pour cela il suffit de cliquer sur l’élément du tableau, sa masse molaire apparaît alors dans le cadre de la calculatrice. La calculatrice dispose aussi d’une touche « Na » pour nombre d’Avogadro pour des formules de chimie. Les formules de chimie sont disponibles en cliquant sur le bouton

Formules de chimie :

QUANTITE DE MATIERE :

n = N/NA n = m/M

Avec :

n : nombre de moles ( en mol ) d'atomes; m : masse des entités ( en g ); M : masse molaire ( en g.mol-1 ) de l'atome ; N : nombre d’atomes ou molécules (sans unité)

CONCENTRATION :

C = n / V

LOI D’AVOGADRO :

P.V = n.R.T

Avec :

R : constante des gaz parfaits = Na.K = 8,314; n : quantité de matière ( mol ) ; V : volume occupé par le gaz ( m3 ) ; P : pression du gaz ( Pa ) ; T : température ( K )
